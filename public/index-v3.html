Common questions.

What's the difference from regular Fedora?

BoringOS is Fedora Atomic with curated defaults. Same base, less configuration. If you want to build from scratch, use Fedora. If you want it to work out of the box, use this.

Can I install my own software?

Yes. Apps install via Flatpak. Dev tools via toolbox containers. The base system stays stable while your workspace stays flexible.

Will my hardware work?

If it works on Fedora, it works here. Most modern hardware is supported. Check compatibility before committing production machines.
