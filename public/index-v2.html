Questions.

Why not just use Fedora?

You can. BoringOS is Fedora Atomic with opinionated defaults and a curated app selection. If you want to configure everything yourself, use Fedora. If you want something that works out of the box, use this.

Can I install my own packages?

Apps via Flatpak. CLI tools via brew or toolbox containers. The base system stays immutable. That's the point.

What about hardware support?

Same as Fedora. If it works on Fedora, it works here. Most hardware just works. Some doesn't. Check before you commit.

Is this for servers?

No. This is a workstation OS. For servers, look at Fedora CoreOS or similar.

What's the catch?

No catch. It's free. It's open source. It's built on Universal Blue and BlueBuild, which are also free and open source.
